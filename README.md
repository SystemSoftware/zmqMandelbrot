# zmqMandelbrot
Using zeroMQ, pyzmq and Python for distributed Mandelbrot crunchin'

You need zeroMQ (http://zeromq.org/), the Python Binding (https://github.com/zeromq/pyzmq) and Python (at least Version 2.7.x / https://www.python.org/downloads/). For Windows, you can find an installation tutorial here: http://www.nico-maas.de/wordpress/?p=1081
